# Django To-Do App

A simple Django-based To-Do web application built to understand core Django concepts such as models, views, templates, and authentication.

This project is intentionally kept simple and beginner-friendly, making it suitable for internship applications.

---

## Features

- User signup, login, and logout
- Add new tasks
- Mark tasks as completed
- Delete tasks
- Each user can see only their own tasks

---

## Tech Stack

- Python
- Django
- HTML
- SQLite (default Django database)

---

## Setup Instructions

Clone the repository
```bash
git clone https://github.com/LibiniP/django-todo-app.git
cd django-todo-app
pip install django
python manage.py makemigrations
python manage.py migrate
python manage.py runserver
Open in browser : http://127.0.0.1:8000/accounts/login/
